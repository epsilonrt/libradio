Cet exemple nécessite les librairies SSDV, SDL et SDL_image.

Pour installer SSDV:
$ git clone git://git.epsilonrt.com/ssdv
$ cd ssdv
$ make
$ sudo make install

SDL et SDL_image sont généralement disponibles dans les dépôts de distib. , par
exemple sous Ubuntu 12.04 :
$ sudo apt-get install libsdl-dev libsdl-image1.2-dev
